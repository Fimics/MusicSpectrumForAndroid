 **项目介绍** 
 
 这是一个Android平台的音乐频谱合集,把音频能量用自定义View给画出来，至于频谱数据的获取或者傅里叶转换之类的不在本篇范畴，网上教程一大把
 
 有需要的童鞋可以拿来一看
